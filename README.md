# sorex_kryptotrade_bot
## Описание проекта:
sorex_kryptotrade_bot это телеграмм-бот отслеживающий min/max значения пар криптовалюта/USD в установленном диапазоне. Телеграмм бот умеет информировать пользователя о достижении пороговых значений, пороговые значения могут быть изменены по желанию пользователя через чат с ботом. Бот работает на удаленном сервере Ubuntu. Доступ к удаленному серверу возможен только по сетевому протоколу SSH.
